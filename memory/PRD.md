# NT - نظام نقاط البيع

## الوصف
نظام POS متكامل لإدارة المخزون والمبيعات والمشتريات باللغتين العربية والفرنسية

## الميزات المنجزة

### إدارة المنتجات ✅
- عرض وإضافة وتعديل المنتجات
- توليد الباركود تلقائياً
- عائلات المنتجات
- تحديث الأسعار الجماعي

### نقطة البيع (POS) ✅
- البيع النقدي والدين والجزئي
- خدمة التوصيل لجميع ولايات الجزائر
- طباعة الفواتير

### إدارة المشتريات ✅
- شراء من الموردين
- الدفع نقداً أو بالدين أو جزئي
- تتبع حسابات الموردين
- تسديد ديون الموردين

### إدارة المال ✅
- الصندوق النقدي
- الحساب البنكي
- المحفظة الإلكترونية
- الخزنة
- تحويل الأموال بين الصناديق

### حصص البيع اليومية ✅ (محدث 2026-02-07)
- **كل موظف يفتح حصته الخاصة**
- فتح حصة يومية مع رصيد افتتاحي
- تتبع المبيعات النقدية وبالدين
- غلق الحصة مع المبلغ الفعلي في الصندوق
- حساب الفرق (فائض/عجز)
- **تبويب "حصصي"** - حصص المستخدم الحالي
- **تبويب "الكل"** - جميع حصص الموظفين (للمسؤول)
- **تبويب "التقارير"** - ملخص إجمالي + تقرير لكل موظف

### عائلات الزبائن ✅ (جديد 2026-02-07)
- تصنيف الزبائن في مجموعات
- إضافة/تعديل/حذف العائلات
- عرض عدد الزبائن في كل عائلة
- البحث في العائلات

### عائلات الموردين ✅ (جديد 2026-02-07)
- تصنيف الموردين في مجموعات
- إضافة/تعديل/حذف العائلات
- عرض عدد الموردين في كل عائلة
- البحث في العائلات

### إدارة المخزون ✅
- المخازن المتعددة
- نقل المخزون بين المخازن
- جرد المخزون
- طباعة الباركود (عدة أشكال)

### التقارير والتحليلات ✅
- صافي الربح مع نسبة هامش الربح
- المبيعات مقابل المشتريات
- أفضل المنتجات مبيعاً
- أفضل الزبائن

### نظام الإشعارات ✅
- تنبيهات المخزون المنخفض
- تذكيرات الديون المستحقة

### الإعدادات ✅
- صلاحيات المستخدمين (admin, manager, user)
- إعدادات الطابعة (UI فقط)
- إعدادات شرائح USB (UI فقط)
- إعادة ضبط المصنع

## APIs الجديدة (2026-02-07)

### حصص البيع اليومية
```
GET  /api/daily-sessions              - قائمة حصص المستخدم
GET  /api/daily-sessions?all_users=true - جميع الحصص (للمسؤول)
POST /api/daily-sessions              - فتح حصة جديدة
GET  /api/daily-sessions/current      - الحصة الحالية للمستخدم
GET  /api/daily-sessions/summary      - تقرير إجمالي + لكل موظف (للمسؤول)
PUT  /api/daily-sessions/:id/close    - غلق حصة
DELETE /api/daily-sessions/:id        - حذف حصة (Admin)
```

### عائلات الزبائن
```
GET    /api/customer-families          - قائمة العائلات
POST   /api/customer-families          - إضافة عائلة
GET    /api/customer-families/:id      - تفاصيل عائلة
PUT    /api/customer-families/:id      - تعديل عائلة
DELETE /api/customer-families/:id      - حذف عائلة
```

### عائلات الموردين
```
GET    /api/supplier-families          - قائمة العائلات
POST   /api/supplier-families          - إضافة عائلة
GET    /api/supplier-families/:id      - تفاصيل عائلة
PUT    /api/supplier-families/:id      - تعديل عائلة
DELETE /api/supplier-families/:id      - حذف عائلة
```
