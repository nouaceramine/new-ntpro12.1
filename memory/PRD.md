## نظرة عامة

## البنية التقنية

### Backend
- Python FastAPI
- MongoDB (Motor async)
- JWT Authentication

### Frontend
- React 18
- Shadcn/UI Components
- RTL Support (Arabic)

### APIs الجديدة
```
POST/GET /api/api-keys          - إدارة مفاتيح API
PUT /api/api-keys/{id}/toggle   - تبديل حالة المفتاح
DELETE /api/api-keys/{id}       - حذف المفتاح

GET /api/recharge/config        - إعدادات المشغلين
POST /api/recharge              - تنفيذ شحن
GET /api/recharge               - سجل الشحن
GET /api/recharge/stats         - إحصائيات الشحن

POST/GET /api/product-families  - إدارة عائلات المنتجات
PUT/DELETE /api/product-families/{id}
```
