# NT - نظام نقاط البيع

## الوصف
نظام POS متكامل لإدارة المخزون والمبيعات والمشتريات باللغتين العربية والفرنسية

## الميزات المنجزة

### إدارة المنتجات ✅
- عرض وإضافة وتعديل المنتجات
- توليد الباركود تلقائياً
- عائلات المنتجات
- تحديث الأسعار الجماعي
- **نشر المنتجات على WooCommerce** (جديد)

### نقطة البيع (POS) ✅
- البيع النقدي والدين والجزئي
- خدمة التوصيل لجميع ولايات الجزائر
- طباعة الفواتير

### إدارة صيانة الهواتف ✅ (جديد)
- تتبع طلبات الصيانة (مستلم، قيد الفحص، قيد الإصلاح، جاهز، مسلم)
- استقبال أجهزة جديدة مع كل التفاصيل
- إدارة مخزون قطع الغيار
- **ربط قطع الغيار مع المخزون الرئيسي** (جديد)
- خصم تلقائي من المخزون عند استخدام القطع
- **إشعارات WhatsApp للعملاء** (جديد) - يتطلب إعداد WhatsApp Business API

### تطبيق PWA ✅ (جديد)
- أيقونة تثبيت التطبيق في الهيدر
- manifest.json للتثبيت على الهواتف
- Service Worker للعمل دون إنترنت
- أيقونات مخصصة (192x192 و 512x512)

### إدارة المشتريات ✅
- شراء من الموردين
- الدفع نقداً أو بالدين أو جزئي
- تتبع حسابات الموردين
- تسديد ديون الموردين

### إدارة المال ✅
- الصندوق النقدي
- الحساب البنكي
- المحفظة الإلكترونية
- الخزنة
- تحويل الأموال بين الصناديق

### حصص البيع اليومية ✅
- كل موظف يفتح حصته الخاصة
- فتح حصة يومية مع رصيد افتتاحي
- تتبع المبيعات النقدية وبالدين
- غلق الحصة مع المبلغ الفعلي في الصندوق
- حساب الفرق (فائض/عجز)
- تبويب "حصصي" - حصص المستخدم الحالي
- تبويب "الكل" - جميع حصص الموظفين (للمسؤول)
- تبويب "التقارير" - ملخص إجمالي + تقرير لكل موظف

### عائلات الزبائن والموردين ✅
- تصنيف الزبائن والموردين في مجموعات
- إضافة/تعديل/حذف العائلات
- عرض عدد الأعضاء في كل عائلة
- البحث في العائلات

### إدارة المخزون ✅
- المخازن المتعددة
- نقل المخزون بين المخازن
- جرد المخزون
- طباعة الباركود (عدة أشكال)

### نظام الإشعارات ✅
- تنبيهات المخزون المنخفض
- تذكيرات الديون المستحقة
- إشعارات العجز/الفائض في الصندوق
- الحد قابل للتخصيص من الإعدادات

### الإعدادات ✅
- صلاحيات المستخدمين (admin, manager, user)
- تغيير كلمات مرور المستخدمين (للمسؤول)
- إعدادات الطابعة (UI فقط)
- إعدادات عامة (حدود التنبيهات)
- **تخصيص صفحة تسجيل الدخول** (شعار، اسم، صورة خلفية) - جديد

## التغييرات الأخيرة (2026-02-09)

### تحسينات جديدة ✅

### وحدة صيانة الهواتف النقالة ✅
تم إنشاء نظام متكامل لإدارة صيانة الهواتف يشمل:

#### الصفحات:
| المسار | الوصف |
|--------|-------|
| `/repairs` | تتبع الصيانة - قائمة الطلبات مع الإحصائيات والبحث والفلترة |
| `/repairs/new` | استقبال جهاز جديد - نموذج تسجيل كامل مع كل التفاصيل |
| `/repairs/parts` | قطع الغيار - إدارة مخزون قطع الغيار مع تنبيهات المخزون المنخفض |

#### APIs الصيانة:
```
GET    /api/repairs           - قائمة طلبات الصيانة
POST   /api/repairs           - إنشاء طلب جديد
GET    /api/repairs/stats     - إحصائيات الصيانة
GET    /api/repairs/{id}      - تفاصيل طلب
PUT    /api/repairs/{id}      - تحديث حالة الطلب
DELETE /api/repairs/{id}      - حذف طلب
```

#### APIs قطع الغيار:
```
GET    /api/spare-parts       - قائمة قطع الغيار
POST   /api/spare-parts       - إضافة قطعة جديدة
GET    /api/spare-parts/stats - إحصائيات المخزون
GET    /api/spare-parts/{id}  - تفاصيل قطعة
PUT    /api/spare-parts/{id}  - تحديث قطعة
DELETE /api/spare-parts/{id}  - حذف قطعة
```

#### حالات طلب الصيانة:
- `received` - مستلم
- `diagnosing` - قيد الفحص
- `waiting_parts` - بانتظار القطع
- `in_progress` - قيد الإصلاح
- `completed` - تم الإصلاح
- `delivered` - تم التسليم
- `cancelled` - ملغي
