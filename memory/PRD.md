# NT - نظام نقاط البيع

## الوصف
نظام POS متكامل لإدارة المخزون والمبيعات والمشتريات باللغتين العربية والفرنسية

## الميزات المنجزة

### إدارة المنتجات ✅
- عرض وإضافة وتعديل المنتجات
- توليد الباركود تلقائياً
- عائلات المنتجات
- تحديث الأسعار الجماعي

### نقطة البيع (POS) ✅
- البيع النقدي والدين والجزئي
- خدمة التوصيل لجميع ولايات الجزائر
- طباعة الفواتير

### إدارة المشتريات ✅
- شراء من الموردين
- الدفع نقداً أو بالدين أو جزئي
- تتبع حسابات الموردين
- تسديد ديون الموردين

### إدارة المال ✅
- الصندوق النقدي
- الحساب البنكي
- المحفظة الإلكترونية
- الخزنة
- تحويل الأموال بين الصناديق

### التقارير والتحليلات ✅ (محدّث 2026-02-06)
- **صافي الربح** مع نسبة هامش الربح
- **المبيعات مقابل المشتريات** رسم بياني مقارن
- إجمالي المبيعات وعدد العمليات
- إجمالي المشتريات وعدد العمليات
- رصيد جميع الصناديق
- مستحقات الزبائن وديون الموردين
- أفضل المنتجات مبيعاً
- أفضل الزبائن
- فترات زمنية: 7 أيام، 30 يوم، 3 أشهر، سنة

### نظام الإشعارات ✅ (محدّث 2026-02-06)
- أيقونة الجرس في الـ Header
- إشعارات تلقائية للمخزون المنخفض
- إشعارات الديون المستحقة (أكثر من 7 أيام)
- إشعارات ديون الموردين
- تحديد الكل كمقروء

### إعدادات النظام ✅
- صلاحيات المستخدمين
- إعدادات الطابعة (حرارية/ليزر/حبر، USB/شبكة/Bluetooth)
- إعدادات شرائح USB (Mobilis/Djezzy/Ooredoo)
- إعادة ضبط المصنع

### شحن الرصيد ✅
- Mobilis, Djezzy, Ooredoo
- أكواد USSD

### إدارة الديون ✅
- ديون الزبائن
- ديون الموردين
- تذكيرات SMS (محاكاة)

## الميزات المكتملة

### المرحلة 1-6: الأساسيات والمتقدمة ✅
- إدارة المستخدمين والمنتجات والزبائن والموردين
- نقطة البيع (POS) مع التوصيل والبيع بالدين
- عائلات المنتجات
- شحن رصيد الهاتف
- إدارة ديون الزبائن مع تذكيرات SMS

### المرحلة 7: الإدارة المتقدمة ✅ (2026-02-06)
- **نظام صلاحيات المستخدمين:**
  - 3 أدوار: admin, manager, user
  - صلاحيات لكل قسم (عرض/إضافة/تعديل/حذف)
  - صلاحيات مخصصة لكل مستخدم
  - إعادة الصلاحيات للافتراضي
- **إعادة ضبط المصنع:**
  - حذف جميع البيانات
  - كود تأكيد للحماية
  - إحصائيات قبل الحذف
- **القائمة الجانبية القابلة للطي:**
  - طي/توسيع بنقرة واحدة
  - حفظ الإعدادات في localStorage
- **توليد الباركود تلقائياً:**
  - صيغة EAN-13 جزائرية (213)
  - رقم تحقق صحيح
  - التحقق من عدم التكرار
- **إضافة عائلة سريعة:**
  - من صفحة إضافة منتج مباشرة
  - بدون مغادرة الصفحة
- **تحديث الأسعار الجماعي:**
  - 3 أنواع: نسبة مئوية، مبلغ ثابت، تحديد قيمة
  - تطبيق على: سعر واحد أو جميع الأسعار
  - فلترة حسب العائلة
  - تقريب تلقائي (5، 10، 50، 100)
  - معاينة التغييرات قبل التطبيق

## صفحات النظام
- `/` - لوحة التحكم
- `/pos` - نقطة البيع
- `/products` - المنتجات
- `/products/add` - إضافة منتج
- `/sales` - المبيعات
- `/recharge` - شحن الرصيد
- `/customers` - الزبائن
- `/customer-debts` - ديون الزبائن
- `/product-families` - عائلات المنتجات
- `/bulk-price-update` - تحديث الأسعار الجماعي
- `/suppliers` - الموردين
- `/employees` - الموظفين
- `/debts` - الديون
- `/cash` - إدارة المال
- `/reports` - التقارير
- `/api-keys` - مفاتيح API
- `/users` - المستخدمين
- `/settings` - الإعدادات (صلاحيات + ضبط المصنع)

## APIs الجديدة
```
# الصلاحيات
GET /api/permissions/roles           - الأدوار والصلاحيات الافتراضية
GET /api/users/{id}/permissions      - صلاحيات مستخدم
PUT /api/users/{id}/permissions      - تحديث الصلاحيات
PUT /api/users/{id}/reset-permissions - إعادة للافتراضي

# النظام
GET /api/system/stats                - إحصائيات النظام
POST /api/system/factory-reset       - إعادة ضبط المصنع

# الباركود
GET /api/products/generate-barcode   - توليد باركود EAN-13
GET /api/products/generate-sku       - توليد SKU

# تحديث الأسعار
POST /api/products/bulk-price-update - تحديث جماعي
GET /api/products/price-preview      - معاينة التغييرات
```
