# ScreenGuard Pro - نظام إدارة المخزون والمبيعات

## نظرة عامة
نظام شامل لإدارة مخزون واقيات الشاشة والمبيعات مع دعم كامل للغة العربية.

## الميزات المكتملة

### المرحلة 1-3: الأساسيات ✅
- تسجيل الدخول والمصادقة (JWT)
- إدارة المستخدمين والمنتجات
- البحث المتقدم، OCR، تنبيهات المخزون
- نقطة البيع (POS)، إدارة الزبائن/الموردين/الموظفين
- الديون والمستحقات، صناديق النقد
- التقارير الرسومية، استيراد/تصدير Excel

### المرحلة 4: API وشحن الرصيد ✅
- واجهة إدارة مفاتيح API
- شحن رصيد الهاتف (Mobilis, Djezzy, Ooredoo, Idoom)
- عائلات/تصنيفات المنتجات

### المرحلة 5: التوصيل والبيع بالدين ✅
- **خدمة التوصيل لكافة الجزائر** (58 ولاية)
- **البيع بالدين** (نقداً/دين/جزئي)
- تصفية المنتجات حسب العائلة

### المرحلة 6: إدارة ديون الزبائن ✅
- صفحة مخصصة لإدارة الديون
- البحث والفلترة والترتيب
- تفاصيل الدين وسجل المدفوعات
- تصدير الديون إلى Excel

### المرحلة 7: تذكيرات SMS ✅ (2026-02-06)
- **نظام تذكيرات SMS للمدينين** (محاكاة - جاهز للربط بمزود حقيقي)
- **تذكير فردي:** اختيار زبائن محددين وإرسال تذكير
- **تذكير جماعي:** إرسال لجميع المدينين بضغطة واحدة
- **قوالب رسائل جاهزة:** ودي، رسمي، عاجل، شكر
- **رسائل مخصصة** مع متغيرات ({customer_name}, {debt_amount})
- **إعدادات التذكير التلقائي:**
  - تفعيل/تعطيل
  - التكرار (يومي/أسبوعي/شهري)
  - يوم ووقت الإرسال
  - الحد الأدنى للدين
- **سجل الرسائل المرسلة**

## الميزات المعلقة/المستقبلية

### P1 - ربط WooCommerce
- مزامنة المنتجات/المخزون/الطلبات مع www.ncr-telecom.com

### P2 - نظام الإشعارات
- أيقونة الجرس مع قائمة منسدلة

### P3 - ربط SMS حقيقي
- تكامل مع مزود SMS محلي جزائري (SMS Algérie, Mobilzone)

## APIs

### SMS System
```
GET /api/sms/templates              - قوالب الرسائل
GET /api/sms/settings               - إعدادات SMS
PUT /api/sms/settings               - تحديث الإعدادات
POST /api/sms/send-reminder         - إرسال تذكير لزبائن محددين
POST /api/sms/send-bulk-reminder    - إرسال جماعي لجميع المدينين
GET /api/sms/logs                   - سجل الرسائل المرسلة
```
