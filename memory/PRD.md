## البنية التقنية

### Backend
- Python FastAPI
- MongoDB (Motor async)
- JWT Authentication

### Frontend
- React 18
- Shadcn/UI Components
- RTL Support (Arabic)

### APIs الجديدة
```
# التوصيل
GET /api/delivery/wilayas           - قائمة 58 ولاية
GET /api/delivery/fee               - حساب رسوم التوصيل

# ديون الزبائن
GET /api/customers/{id}/debt        - دين الزبون وسجل المدفوعات
POST /api/customers/{id}/debt/pay   - سداد دين
GET /api/debts/summary              - ملخص جميع الديون

# API Keys
POST/GET /api/api-keys              - إدارة مفاتيح API
PUT /api/api-keys/{id}/toggle       - تبديل حالة المفتاح

# شحن الرصيد
GET /api/recharge/config            - إعدادات المشغلين
POST /api/recharge                  - تنفيذ شحن
GET /api/recharge/stats             - إحصائيات الشحن
```
